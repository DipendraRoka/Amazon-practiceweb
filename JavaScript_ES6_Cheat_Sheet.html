JavaScript ES6+ Cheat Sheet

map()

const doubled = [1,2,3].map(n => n * 2);

filter()

const even = [1,2,3,4].filter(n => n % 2 === 0);

find()

const user = users.find(u => u.id === 2);

reduce()

const total = [1,2,3].reduce((sum,n)=>sum+n,0);

every()

const ok = [2,4,6].every(n => n % 2 === 0);

some()

const hasEven = [1,3,4].some(n => n % 2 === 0);

Promise

const promise = new Promise((resolve,reject)=>{
  resolve("Success");
});
promise.then(console.log).catch(console.error);

async / await

async function getData(){
  try{
    const res = await fetch("https://jsonplaceholder.typicode.com/users");
    const data = await res.json();
    console.log(data);
  }catch(err){
    console.error(err);
  }
}

Promise.all()

const [users,posts] = await Promise.all([
 fetch('/users'),
 fetch('/posts')
]);

Quick Summary

MethodPurpose
map()Create a transformed array
filter()Keep matching items
find()Return first match
reduce()Combine into one value
forEach()Loop through items
every()All items match?
some()Any item matches?
PromiseHandle async work
async/awaitCleaner async syntax
